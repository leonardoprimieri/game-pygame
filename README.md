## The mandalorian ship

Como fiquei sabendo que o professor Marcão gosta do universo de StarWars, resolvi fazer o game com a temática do mesmo. 
Baseado na série The Mandalorian, o objetivo do game é desviar dos lasers inimigos com sua nave. 
Jogo de mecânica simples, mas desafiante de ser implementada. 
Depois dessa descrição, a única coisa que te resta é jogar! Bom jogo, e que a força esteja com você!
